## Project Description

Create a C# Windows Forms Application that implement Basic Numerical Operations

## Key Components

*   Addition (+)
    
*   Subtraction (-)
    
*   Multiplication (\*)
    
*   Division (/)
    
*   Exponentiation (^)
    
*   Roots (√)
    
*   Modulo (%)
    
*   Absolute Value (|ABS|)
    

\-"+/-" NEGATE button to implement negative number

*   "DEL" button to erase last input digit
    
*   "AC" button to reset input
    
*   "=" button for calculation
    

## Key Features

Implement the whole program using Pure Functions for Mathematical Functions and String Manipulation, for Easier Testing, Improve Readability and Resuability

## Tech Stack

100% C#
